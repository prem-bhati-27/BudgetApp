Text input on the dark input surface; the teal border appears on focus.

```jsx
<Input icon="search" placeholder="Search transactions" />
<Input type="amount" prefix="₹" placeholder="0" />
```

- `type="amount"` right-aligns the value in Space Mono — use for money entry.
- `prefix` renders a fixed mono symbol (e.g. "₹") before the field.
